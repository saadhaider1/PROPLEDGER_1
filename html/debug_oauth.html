5. Setup Instructions

To get real Google OAuth working:

  1. Go to Google Cloud Console
  2. Create a new project or select existing
  3. Enable Google+ API
  4. Create OAuth 2.0 credentials
  5. Add redirect URI: http://localhost/PROPLEDGER/auth/oauth_callback.php?provider=google
  6. Copy Client ID and Client Secret to config/oauth_config.php
